# 简介

**个人配置文件**

- 系统环境：archlinux
- 桌面环境：i3wm、niri、xfce4
- 桌面组件：polybar、plank,rofi、waybar、wlogout...
- compositor：picom
- p.s. i3很久不用了。
- old/里是一个以前参考[polybar themes](https://github.com/adi1090x/polybar-themes)的polybar配置

## 字体

- 中文：苹方
- 英文：inter-font
- 等宽：ttf-jetbrains-mono-nerd

```sh
yay -S inter-font ttf-jetbrains-mono-nerd otf-apple-pingfang ttf-apple-emoji
```
